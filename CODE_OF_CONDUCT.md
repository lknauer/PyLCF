# Code of Conduct

This project adopts the
[Contributor Covenant](https://www.contributor-covenant.org/version/2/1/code_of_conduct/),
version 2.1.

In short: be respectful and constructive. Harassment or discrimination of any
kind is not tolerated.

To report unacceptable behavior, please contact the maintainer (Lukas Knauer,
AG Schünemann, RPTU Kaiserslautern-Landau). Reports will be handled
confidentially.
